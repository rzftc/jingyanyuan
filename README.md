# 监管能力分析系统

一个基于 MATLAB 的框架，用于监管能力的分析与预测。

## 目录结构

核心代码位于 `10regulatorycapacity` 文件夹，模块简介：

| 模块路径             | 功能                |
|-------------------|-------------------|
| **0input_data**  | 数据存储与预处理       |
| **1initialize**  | 参数和环境初始化       |
| **2AC / 3EV**    | 核心计算模块         |
| **4analysis**    | 基础分析功能         |
| **5userUncertainties** | 用户不确定性建模      |
| **6SensAnalyses** | 灵敏度分析          |
| **7main**         | 主程序入口           |
| **9predict**      | 预测模块            |
| **10spaceRelation** | 空间关系计算         |
| **11stackburg**   | Stackelberg 博弈模型 |

#### 模块介绍与用法说明
| 模块路径 | 简介 | 用法 |
|---------|-----|------|
| **0input_data** | 数据存储与预处理 | 将数据文件放入目录，通过模块完成数据初步处理 |
| **1initialize** | 系统初始化 | 初始化分析参数及运行环境 |
| **2AC / 3EV** | 核心计算 | 执行分析所需主要计算任务 |
| **4analysis** | 数据分析 | 生成分析结果和图表 |
| **5userUncertainties** | 不确定性建模 | 定义用户输入条件，探究系统对参数变动的响应 |
| **6SensAnalyses** | 灵敏度分析 | 对关键维度执行灵敏度分析实验 |
| **7main** | 主程序入口 | 项目核心运行主目录，直接运行完成全流程 |
| **9predict** | 预测计算 | 模拟未来性能变化趋势 |
| **10spaceRelation** | 空间计算 | 分析不同变量的空间分布关系 |
| **11stackburg** | 博弈模型 | 解算多参与方博弈问题 |

## 快速开始

1. 安装 MATLAB。
2. 进入 `10regulatorycapacity/7main` 目录。
3. 运行主脚本进行分析。
